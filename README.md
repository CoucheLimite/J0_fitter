# J0_fitter

This is a program written in python to caculate the determine the recombiation current at a surface of a solar cell from
quasi steady state photoconductance measurements. The program currently accepts inputs from a Sinton Instruements excel file. 

## Installation

install 2 of my other packages:
  1. Semiconductor
  2. PV_analyser
  
The just dowload and run main.py with python 3.5.
